Messages API

This is messages API to share data with other applications. It is divided to two main parts Guests vs Registered Authors.

Guests can do:

  • List all of the messages and their inner comments. Three messages per page. 
              http://localhost:8000/api/version?(v1|v2)/messages/
  • Display only one detailed message with its inner comments. 
              http://localhost:8000/api/version?(v1|v2)/message/(message_slug)
  • Search for a specific keyword in all of the messages list. Search fields: message-title, message-body. 
              http://localhost:8000/api/version?(v1|v2)/messages/?q=keyword

Registered authors can do:

  • List all of the messages and their inner comments. Three messages per page. 
              http://localhost:8000/api/version?(v1|v2)/messages/
  • Share new messages. You have to an author to create a new message. 
              http://localhost:8000/api/version?(v1|v2)/messages/
  • Display only one detailed message with its inner comments. 
              http://localhost:8000/api/version?(v1|v2)/message/(message_slug)
  • Update their own messages. You have to be the author of that message so you can update it. 
              http://localhost:8000/api/version?(v1|v2)/message/(message_slug)
  • Delete their own messages. You have to be the author of that message so you can delete it. 
              http://localhost:8000/api/version?(v1|v2)/message/(message_slug)
  • Search for a specific keyword in all of the messages list. Search fields: message-title, message-body. 
              http://localhost:8000/api/version?(v1|v2)/messages/?q=keyword
